Thinkphp-react-admin
===============

> 运行环境要求PHP7.2+，兼容PHP8.1

## 安装依赖

### php依赖

后端启动

```
composer install  
php think run
```

### 前端开发依赖

前端启动

```
npm run install
npm run start
// or
yarn
yarn start
```
